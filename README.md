## 🎮 주요 기능 설명
### 소비자용 서비스
1. 로그인 화면
- 이메일과 비밀번호를 입력할 수 있다.
- [로그인] 버튼을 누르면 로그인 처리가 이뤄진다.
- 아이디 저장, 자동 로그인을 체크할 수 있다.
- 아이디, 비밀번호 찾기를 할 수 있다.
- [회원가입] 버튼이 있다.

![img_8.png](img_8.png)

- [아이디 중복 확인] 버튼을 클릭하면 중복되는 아이디가 있는지 체크한다.
